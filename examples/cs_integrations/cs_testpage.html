ETH

The ETH integration is a link-based integration (PSVS).
Users have the option to search for a term in the ETH library catalog and then click on the "Open Knowledge Maps" link to get a knowledge map for the search term on the OKMaps website.
ETH has made two outlinks available, on where the search term is passed on to BASE and on to PubMed.
The integration is also not using the new CS search box, but the old one (embed=true is not set).
Other parameters are default parameters.
  • BASE: Chemical
  • PubMed: Chemical
  • BASE: failpage
  • PubMed: failpage

GoTriple

The GoTriple integration is a link-based integration (PSVS) with embedded iFrames.
Users have the option to search for a term in the GoTriple catalog and then create Knowledge Maps or Streamgraphs for the search term.
The GoTriple integration using the new CS search box (embed=true is set), and visualisations are embedded in iFrames in the GoTriple website.
GoTriple makes regular use of date range parameters (from, to), language parameters (lang_id), and restricts searches to their content (repo=fttriple).
  • Knowledge Map: (fair or "open access") and publishing
  • Streamgraph: (fair or "open access") and publishing
  • Failpage Knowledge Map: asdfdasdfda
  • Failpage Streamgraph: asdfdasdfda

UNIL

The UNIL integration is a link-based integration (PSVS).
CS integrators have prepared predefined queries which are used on topical overview pages.
Users can click on a link a get to a Knowledge Map for a predefined query on the OKMaps website.
The UNIL integration is not using the CS search box, but the old one (embed=true is not set).
UNIL makes use of advanced queries incl. Boolean operators and nested queries, and restricts searches to their content (repo=ftunivlausanne).

AUSSDA

The AUSSDA integration is an iframe-based integration of the search box (SBCS).
The integration is restricted to the AUSSDA repository (repo=ftaussda) and min_descsize is set to 0.
The show_service and show_sorting parameters are set to false.
The search box has Dataset and "Other/Unknown material" preselected as document types.
